What is TypeScript?

TypeScript is a typed superset of Javascript that compiles into plain, idiomatic Javascript

Cool, but what does that even mean?

  • TypeScript compiles into readable Javascript
  • Valid Javascript === valid TypeScript
  • TypeScript adds optional static typing
  • It brings together the best of typed languages and the dynamic nature of JS
  • It prevents many classes of issues before runtime
